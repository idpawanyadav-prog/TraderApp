    API Access

    Allow external applications to fetch data through TraderApp using a secret API key.

    Enable API Connection Expose a public REST API on this machine
    Available Endpoints No authentication required — just enable the toggle above
    Technical Indicators Append indicator values to API responses when TA=true
    Usage: append TA=true to any historical API call.
    Example: /public/api/5paisa/historical?symbol=RELIANCE&interval=15&TA=true
